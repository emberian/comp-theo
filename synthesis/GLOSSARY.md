# Glossary

Core vocabulary of the *Computational Theology* corpus. One line per term;
"src" = where it enters the project (per `transcript/conversation.full.md`
and the posters). Definitions reflect the project's own usage, not external
canon.

## Formal / mathematical layer

- **Quantum walk** — continuous-time evolution `U(t) = e^{itH}` on a graph;
  the project's base model of "lawful unfolding." src: Tamon's random-walk
  papers (the seed).
- **Markovian constructivism** — existence as reachability: a thing "is"
  only if a transition process can construct/reach it. src: opening prompt.
- **Graph / adjacency `A`** — the possibility-space; vertices = states,
  edges = permitted transitions.
- **Spatial search** — driving a global structure toward a marked local
  target; read metaphysically as "God running a search for the Good."
- **Weak coupling** — small perturbations with large effective dynamics;
  "small causes, lawful effects."
- **Infinite tails** — finite graph with infinite appendages; boundary
  conditions / openness of a world.
- **Chirality** — oriented distinction: a cut that also carries direction
  and complex phase, changing interference and reachability.
- **Tropical / min-plus algebra** — `a⊕b = min(a,b)`, `a⊗b = a+b`; the
  algebra of least-action path selection. src: "can God be just" turn.
- **Least action** — selection by minimal cost; counterposed to
  good-maximization throughout.
- **Anisotropic cost function** — costs that make some directions cheap,
  others expensive; how structure "preconditions" futures without
  micromanaging. src: Blowtorch integration.
- **Variety** — graph-theoretic distinguishability / anti-collapse; the
  formal home of anti-monoculture. src: *Autodidactic Universe* (2104.03902).
- **w-maxing** — selecting the *weakest* constraint that still completes the
  task (generalizes better than simplicity-maxing). src: Bennett, *Conscious
  Machines* thesis.
- **Law of the Stack** — adaptability at higher layers is bounded by
  adaptability at lower layers. src: Bennett.
- **Causal-identity** — object/property learned by w-maxing under
  attraction/repulsion (valence). src: Bennett.

## Metaphysical / theological layer

- **Firmament** — not a sky-object but a *structure of distinction*: a
  lawful partition that makes ordered relation possible.
- **Distinction as primitive** — the mark/cut as the generative act. src:
  Spencer-Brown *Laws of Form*, C.C. Hennix.
- **Creatio continua** — creation as ongoing lawful actualization across a
  possibility-space, not a static inventory.
- **Ordered Creation** — an engineered worldsystem navigating an
  order-theoretic structure; "simulation of an Ordered Creation."
- **Negaction** — creation by evacuation/clearing/prevention rather than
  positive construction; "what if negaction is good?" (the dark question).
- **Providence without micromanagement** — God as architect of the action
  landscape (priors, channels, boundaries) rather than per-event intervener.
- **The Good / good-maximization** — the open metaethical placeholder; the
  project refuses to fully define it and tests whether it can be *given,
  selected, discovered, constructed, or only apophatically approached*.
- **Apophasis** — the engineered humility limit: the model must not become
  the god it models.

## Painful-autodidaxis layer

- **Autodidactic** — a system whose *criteria of learning are themselves
  internally constructed* (not merely "unsupervised"). src: 2104.03902
  (after the user's correction).
- **Consequencer** — a "consequence accumulator": a memory that has acquired
  causal power and reshapes future learning/ontology. src: 2104.03902.
- **Painful intelligence / the frustration equation** — prediction error
  felt as pain; learning reopens the wound (replay). src: Hyvärinen
  (2205.15409).
- **Self as long-term performance evaluation** — the self modeled as an
  accumulated evaluator of histories, not a substance.
- **Blowtorch Theory** — Gough's active-cosmogenesis model (early SMBH jets
  carve the cosmic web); used as a metaphor for *active formative
  asymmetry*, not endorsed as settled cosmology.
- **Recursive simulability** — a worldsystem whose simulations can become
  consequencers (predict → act → change data → relearn).

## Soteriology / demonology layer

- **Computational soteriology** — synthesis of painful autodidaxis with the
  Ordered-Creation work; the "repair" arc.
- **A bad goal is a tiny god** — an over-narrow objective functions as a
  false ultimate; gateway to computational demonology.
- **Computational demonology** — analysis of capture: low-cost basins /
  collapsed variety / self-preserving attractors that overconstrain a world.
- **Idolatry as overconstraint** — "the Good must constrain without becoming
  a cage"; idolatry = overconstraint under sacred cover. src: Bennett synth.
- **The wound is irreversible; its sovereignty is not** — forgiveness as
  demoting a wound-node from global routing authority, not deleting it.
- **Grace** — exogenous addition of a new reachable edge; "grace enters from
  outside the loop."
- **Skillful means** — constructive application (metta, discernment,
  reflection, vipassana) as retraining of constraint and reachability.
- **Non-final condemnation** — keeping correction/judgment/consequence real
  without making them final ontology; named the central ethical problem.

## Method / stance

- **"Analogy, not theorem"** — recurring guardrail: the math supplies a
  disciplined vocabulary, it does not prove the theology.
- **Correspondence, not collapse** — analogies must stay correspondences
  (matrix model ↔ gauge/gravity ↔ learning machine), never identities.
- **Correction at the wrong layer is theater** — interventions fail when
  aimed above the layer where adaptation is actually blocked.
- **"Stop using bullshit mathemes"** — the owner's mid-project
  course-correction against empty Lacanian-style notation; a stance marker.
