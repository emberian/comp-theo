BLOWTORCH-SHAPED DESIRE
World-formation does not merely shape matter; it shapes what becomes attractive.

1. CORE IDEA
Energetic structure-formation carves channels, voids, and boundaries.
The world becomes anisotropic: some directions flow, others resist.
These patterns precondition what is easy, possible, and ultimately desirable.

2. ANISOTROPIC COST
Some directions become cheap; others expensive.
LOW COST (Expensive to traverse) ... HIGH COST
Low Cost / High Cost

3. PRIVILEGED CORRIDORS
Low-cost pathways attract traffic, attention, and desire.
MAP FLOWS: Main Corridor (lowest cost), Secondary Path (higher resistance), Dead End / Void (impassable). Traffic, Resources, Attention, Desire.

4. CARVED TEMPTATION
The environment preconditions what agents will find rewarding and reachable.
Desire is not only discovered—it is scaffolded by what is easy to approach.
Raw Value Landscape (potential) -> World Carving (Cost Structure) -> Accessible Reward Peaks (realized)
desire(x) ~ value(x) - path resistance(x)
Low Resistance => Attractive ; High Resistance => Suppressed Value ; Becomes Unattractive

5. EXTERNALITIES
Privileged channels can starve alternatives and link costs elsewhere.
Channel Costs (Visible Benefits): Rapid Scale, Reliable, Focus.
Hidden Costs (Displaced Risk): Lost Diversity, Initial Risk, Coarse Suffering.

6. MORAL AMBIGUITY
Carved intensity can heal or harm, create or diminish.
HEALING FLOW: Connection, Creativity, Stability, Direction.
DOMINANT FLOW: Concentration, Extraction, Capture, Depletion.
Same structure. Different orientation. Different fruit.

7. INTERIOR CORRELATE
Agents inside such a world inherit shaped expectation landscapes.
EXPECTATION FIELD (What feels probable): High Expectation, Low Expectation.
AGENT SOURCE: What's nearby, What's rewarded, What's normal, What's worth trying.
INNER DESIRE ATTRACTOR MAP (What becomes magnetic): Strong Attractor, Weak Attractor, Repelled / Aversive.
Thus, character and culture co-evolve with cost-shaping.

When the world carves the path, it also half-writes the wanting.
