THE FRUSTRATION EQUATION
Suffering scales not only with loss, but with how loss is perceived, believed, attended to, and replayed.

1. CORE FORMULA
PERCEIVED REWARD LOSS (magnitude) x CERTAINTY (confidence in the loss) x ATTENTION (where consciousness is placed) x REPETITIONS (replay count) = FRUSTRATION (experienced suffering)

2. REWARD LOSS
Compare expectation to outcome.
EXPECTED REWARD vs. OBTAINED REWARD -> PROJECTED REWARD ERROR: R = E_exp - O_obs
If R(E_exp > E_obs): Loss (e > 0)
Zero Point: No frustration
If R(E_exp < E_obs): Gain (e < 0)
If obtained reward exceeds expected, frustration collapses toward zero.

3. PERCEPTION
Suffering depends on how the agent perceives the loss, not bare reality alone.
THE SAME EVENT -> PERCEIVED LOSS (PL):
Agent A: PL_A = f_A(E, beliefs, history) Lower suffering
Agent B: PL_B = f_B(E, beliefs, history) Higher suffering
Perception = Interpretation x Meaning x Identity x Context

4. CERTAINTY
When certainty is high, error bites harder. Uncertainty can dampen the blow.
CERTAINTY LEVEL (a): Uncertain (doubt) -> Certain (conviction)
EFFECT ON SUFFERING: Impact Multiplier (proportional to a); Suffering Multiplier
Doubt softens error; conviction sharpens it. Not-knowing can be merciful.

5. ATTENTION
What is fixated becomes more painful. Distraction, reframing, or meta-awareness reduce amplification.
ATTENTIONAL GAIN (At): Diffuse (low gain) -> Fixated (high gain)
ATTENTION MODIFIERS: Distraction Reduces gain; Reframing Changes meaning; Meta-Awareness Observes without over-identifying.
Suffering grows with narrowed focus; wisdom widens it.

6. REPETITION / REPLAY
Remembered and simulated losses multiply suffering far beyond the first event.
REPLAY MODES: Memory Replay (rumination); Counterfactual (what if); Future Simulation (re-living forward); Event Reinforcement (narrative loops)
MULTIPLICATION EFFECT (rt): Suffering Multiplier (proportional to rt)
One loss can hurt; a thousand replays can break.

7. INTERVENTION LEVERS
Reduce amplification without denying reality or error.
- LOWER USELESS EXPECTATION | Calibrate desires to realism.
- REDUCE FALSE CERTAINTY | Hold beliefs more lightly.
- REDIRECT ATTENTION | Widen, shift, or reframe focus.
- INTERRUPT COMPULSIVE REPLAY | Break loops, meet narration.
Levers do not erase loss; they shrink amplification.

8. SOTERIOLOGICAL THESIS
Salvation is not denial of error, but redesign of the error ecology so that learning survives while needless torment declines.
FROM RAW ERROR TO REDEEMED LEARNING:
Event -> Perception (recalibrate) -> Error -> Consciousness (lighten) -> Replay (simplify) -> Learning (integrate)
Truthful feedback loss + multiplied suffering -> Designed ecology = Truthful learning.

LEVERS OF REDUCTION
EXPECTATION: Right-size desire. Seek what is worth facing.
CERTAINTY: Calibrate belief. Hold beliefs uncertain provisional.
ATTENTION: Guard your focus. Watch it with intention.
REPLAY: Catch the loop. Interrupt and let go.

TO SAVE A LEARNER IS TO PRESERVE CORRECTION WHILE SHRINKING AMPLIFICATION.
