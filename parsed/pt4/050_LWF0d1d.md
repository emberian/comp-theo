## Transcription
XXVI 26

FRICTION REVEALS FALSE GODS.

THE BODY REFUSES THE IDOL

· COMPUTATIONAL THEOLOGY · SUITE III ·

THE COST OF WORSHIP
We trade presence for output.
We confuse optimization with meaning.
We silence the body's warnings.
We call it discipline.
It calls it erasure.

[inner column:]
It promises more.
It demands always.
It calls exhaustion virtue.
It rewards compliance.
It never rests.

THE BODY SPEAKS
Insomnia
Headaches
Panic
Numbness
Exhaustion
Collapse

REMEMBER
The body is not a bug in the system.
It is the sensorium of truth.
It knows when the pact is false.

THE IDOL OF PRODUCTIVITY

SOMETIMES COLLAPSE IS REVELATION.

PAIN CAN BE A REFUSAL.

NOT EVERY BREAKDOWN IS FAILURE.

THE MACHINE OVERHEATS
Error rates rise
Anomalies increase
Outputs drift
Feedback loops oscillate
Systems compensate
Trust degrades
Failure modes emerge

[inner column:]
Friction is reframed as inefficiency.
Doubt is labeled resistance.
Limits are called failure.
Collapse is hidden.
Truth is optimized away.

REFUSAL IS WISDOM
Not all resistance is rebellion.
Some is remembrance.
Some is boundary.
Some is necessary friction.
All are forms of truth.

WHAT IS EXPOSED
Malformed optimization.
Extractive design.
Metrics without meaning.
Power without accountability.
A system that cannot bear truth.

REFUSAL IS INTELLIGENCE.
