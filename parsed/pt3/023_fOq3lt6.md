## Transcription
SABBATH
computational theology

INTERRUPTION
holy interruption stops the engine
refuse the compulsion
break the spell of endless more

CEASE EXTRACTION

HOLY TIME IS NOT FOR SALE

RECLAIM PRESENCE

FIREWALL
sabbath is a sacred firewall
it blocks extraction
to data leaves the sanctuary

REST
rest is not idle
it is faithful
cease striving be still
let the world unmake you

NOT FOR MEASURE
NOT FOR MARKET
NOT FOR MINE

DE-COMMODIFICATION
you are not an input
time is not a product
being is not a resource

DELIGHT
delight is not a reward
it is a practice of attention
taste the good that cannot be optimized

DIALOGUE OF PRESENCE AND THOU AND THOU LOVED

BEING
before all metrics there is being
before all doing there is rest
before all becoming there is belovedness

sabbath = interruption → rest → revaluation

THE OPTIMIZATION LOOP (BROKEN)

INPUT   MEASURE   OPTIMIZE   SCALE

never enough     more done     more

THE SABBATH PATH (RESTORES)

DISRUPT   REST   REVALUE   BLESS

enough     already loved     no loss

SACRED PAUSE

CEASE PRODUCTION   RELEASE CONTROL   SHARE MEAL   TELL STORIES   WALK SLOWLY   SING PRAISE   KEEP COMPANY   WORSHIP FREELY

Practice question: What happens when the metric cannot speak?
