## Transcription
VOWS CREATE
LOW-COST RETURNS
computational theology

A VOW IS A PORTABLE PATH
BACK TO THE GOOD.
IT CAN BE RE-INSTANTIATED
AT ANY SCALE.

VOW
a freely chosen orientation to the good written in word, deed, or symbol.
light enough to carry, strong enough to call back.

RECURSION
the vow renews the path each time you return.
each return lowers the cost of the next return.

SCALE
a vow works at any scale.
in a breath in a day in a life in a city in a culture

RETURN
coming back to the good realigns attention, energy, and will.
not perfection — reconnection.

ORIENTATION
the vow sets direction before the decision.
it points you home when clarity is low.

FIDELITY
small, faithful returns compound into strength.
the vow keeps trust with the good.

RENEWAL return renewed
DEVICE return given
TRUTH return aligned
REPAIR return mended
COURAGE return strengthen
GRATITUDE return awakened
WISDOM return

WHEN YOU START, THE VOW STANDS.
WHEN YOU REMEMBER, THE PATH OPENS.
WHEN YOU RETURN, THE GOOD GROWS.

vow = orientation → return → reinstantiate

THE VOW PROCESS: A PORTABLE PATH
1. FORGET attention drifts  2. REMEMBER the vow calls back  3. RETURN step back on the path  4. RENEWAL realign to the good  5. REPEAT the path deepens
form — re-orient — choose — realign — act

TYPES OF VOWS
WORD spoken — DEED enacted — SYMBOL embodied — WITNESS made before others — RETURNING vow renewed

VOW ATTRIBUTES
CLARITY  DEVOTION  SIMPLICITY  SEVERANCE  CONTEXT  ACKNOWLEDGMENT  FLEXIBILITY  LOVE
